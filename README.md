# Amazon Prime Video Hold Space Speedup

* Script basically makes Amazon Prime Video players function like YouTube player *

A Tampermonkey userscript for Amazon Prime Video
It lets you hold **Space** to temporarily speed up playback, tap **Space** to play/pause, and keeps controls visible when paused

## File
`Amazon_Prime_Video_HoldSpaceSpeedup.user.js`

## Overview
- Hold Space = speed up
- Tap Space = play/pause
- Default boost speed = 2x
- Custom speed up to 3x
- Controls hide quickly while video is playing

## Install
Install Tampermonkey, then open the `.user.js` file and add it to your browser.

## Notes
Made for Amazon Prime Video in desktop browsers (should work in any)
